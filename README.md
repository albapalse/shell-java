# Shell in Java

This repository contains my Java implementation of a minimal Unix-like shell.

The project was built as part of the CodeCrafters “Build Your Own Shell” challenge. The goal is to understand how command-line shells work internally, including command parsing, built-in commands, PATH lookup, and process execution.

## Features

- Interactive shell prompt
- Handling of unknown commands
- Built-in command support
- Execution of external programs
- Basic command parsing

## Tech Stack

- Java
- Git / GitHub
- CodeCrafters test-driven challenge environment
